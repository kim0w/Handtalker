프로젝트 개요

이 프로젝트는 수어(한국수어) 학습자를 위한 AI 기반 퀴즈 학습 앱입니다.
사용자는 수어 영상을 보고 정답을 맞히는 퀴즈를 풀며 수어 단어를 자연스럽게 익힐 수 있습니다.
퀴즈가 끝나면 점수와 날짜가 자동으로 저장되어, 사용자는 자신의 학습 기록을 확인할 수 있습니다.

대상 사용자

수어를 처음 배우는 초보자
한국수어 단어를 반복 학습하고 싶은 학습자
수어 강의 보조용 앱이 필요한 교육자
영상 기반 퀴즈 앱을 연구/개발하려는 개발자

프로젝트의 목적

이 프로젝트는 수어 학습의 접근성을 높이고,
누구나 쉽게 수어 어휘를 익힐 수 있는 인터랙티브 학습 도구를 만드는 것을 목표로 합니다.
특히 영상 기반 퀴즈 + 점수 기록 시스템을 결합하여
학습자의 진도 관리와 반복 학습 효율을 극대화합니다.

핵심 기능 요약

1. 수어 영상 기반 퀴즈 제공
총 96개의 수어 단어를 기반으로 구성된 객관식 문제.
각 단어별 한국수어사전 공식 영상(WebM) 자동 재생.
옵션(보기) 순서가 매 퀴즈마다 자동 랜덤 셔플되어 난이도 증가.

2. 퀴즈 점수 자동 저장 시스템
퀴즈 완료 시 점수 + 날짜가 자동 저장됩니다.
데이터를 안전하게 관리하기 위해 SharedPreferences(JSON) 사용.
앱을 재시작해도 기록 유지.

3. 나의 퀴즈 기록 화면
지금까지 푼 모든 퀴즈 점수를 리스트 형태로 확인 가능.
날짜별 진행 상황을 쉽게 파악할 수 있어 학습 동기 부여.
필요 시 전체 기록 초기화도 가능.

4. 유지보수를 위한 구조 설계
96개 문제는 코드가 아님 → 분리된 JSON 파일로 관리.
Flutter에서 JSON을 불러와 문제를 구성하기 때문에
문제가 추가되거나 변경될 때 앱을 수정할 필요 없음.

기능 특징
퀴즈 완료 시 자동 저장
퀴즈가 종료되면 점수와 날짜가 SharedPreferences에 자동 저장됩니다.

모든 기록을 JSON 배열 형태로 관리
여러 회차의 기록을 누적 관리합니다.

날짜는 ISO 8601 형식으로 저장
→ 2025-12-07T12:33:11.842Z 형태
→ 정렬, 가공, 변환이 쉽습니다.

기록 조회 화면에서 리스트 형태로 출력
ListView 기반으로 날짜 + 점수를 깔끔하게 보여줍니다.

기술 스택 (Tech Stack)
Flutter 3.x
Dart
Video_Player (수어 영상 재생)
SharedPreferences (퀴즈 기록 저장)
JSON 구조 문제 파일 로드
StatefulWidget 기반 퀴즈 UI

구성 요소
파일명	역할
